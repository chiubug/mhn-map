# MHNbiome
get habitat from s2cells

# Requirement
* s2sphere
* python-dateutil
 
# Installation
python -m pip install s2sphere
python -m pip install python-dateutil

# Usage
1. open 'settings.py'
2. edit datetime and ractangle
3. execute MHNbiome.py
4. upload 'MHNbiome.kml' to GoogleMap 'My Map' Layer
